This is a basic font that implements WoodyLing's phonetic alphabet.

So it takes a bit of relearning to think about the sound rather than which letter you'd usually write.

It uses only the lowercase latin letters and 13 ligatures to represent the 39 phonemes that Woody came up with.

The leftover letters were repurposed to sounds that use the same letter as another sound:
- c (/ai/ in why, y is used for the y sound in you)
- q (schwa, like e in problem; e is used for the e sound in bet)
- x (the s sound in usual; s is specifically for the s sound in sing)

Since the letters are specific to certain sounds, there are 13 sounds that need to be split off, and they are handled with ligatures:
- ee (as in beet)
- ch (as in child)
- ou (as in you)
- th (voiced form, as in think)
- dh (unvoiced form of th, as in the)
- sh (as in shin)
- oo (as in wood)
- ai (as in bait)
- ng (as in sing)
- oa (as in boat)
- ough (as in bought)
- ow (as in how)
- oy (as in joy)
